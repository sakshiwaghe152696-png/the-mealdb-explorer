#  TheMealDB Explorer  
# A Full-Stack Recipe Search Application (Finfactor Assignment)

TheMealDB Explorer is a mini full-stack project built as part of the Finfactor Technologies coding assignment.  
It consists of:

- A Node.js + Express backend that wraps TheMealDB public API with caching.
- A modern, responsive frontend built using HTML, CSS and JavaScript.
- Features: search meals, browse by category, random meal, and detailed recipe viewer.

---

# Assignment Requirements — How This Project Meets Them

# Backend (Web Service API)
The backend is built using Node.js + Express, and provides simplified REST endpoints for the UI.

It includes:
- In-memory caching system  
- Cache expiry (5 minutes)  
- Maximum cache size (100 entries)  
- REST-style endpoints  
- Runs locally on `http://localhost:5000`  
- Fetches data from **TheMealDB API (key = 1)**

# Frontend (UI Layer)
Built using HTML + CSS + JavaScript, featuring:
- Recipe Search
- Category Browser
- Random Meal Generator
- Detailed Recipe View
- YouTube video embed
- Fully responsive design  
- Modern pink–lavender theme for a beautiful UI
